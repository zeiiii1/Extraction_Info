========================
The QUAERO French Medical Corpus, CLEF eHealth release
========================

The QUAERO French Medical Corpus CLEF e-Health release comprises a subset of the QUAERO French Medical corpus, as follows:

Training data (used in CLEF eHealth 2015 task 1b as training data): 
- MEDLINE folder: 833 MEDLINE titles, annotated in the BRAT stand-off annotation format, and 3 BRAT configuration files
- EMEA folder: 3 EMEA documents, segmented into 11 files, annotated in the BRAT stand-off annotation format, and 3 BRAT configuration files  

Development data  (used in CLEF eHealth 2015 task 1b as test data and in CLEF eHealth 2016 task 2 as development data): 
- MEDLINE folder: 832 MEDLINE titles, annotated with normalized entities in the BRAT stand-off annotation format, and 3 BRAT configuration files
- EMEA folder: 3 EMEA documents, segmented into 12 files, annotated with normalized entities in the BRAT stand-off annotation format, and 3 BRAT configuration files

Test data (Used in CLEF eHealth 2016 task 2 as test data): 
- MEDLINE folder: 833 MEDLINE titles, annotated in the BRAT stand-off annotation format, and 3 BRAT configuration files
- EMEA folder: 4 EMEA documents, segmented into 15 files, annotated in the BRAT stand-off annotation format, and 3 BRAT configuration files



This release of the QUAERO French medical corpus, CLEF e-Health version comes in the BRAT stand-off annotation format (http://brat.nlplab.org/standoff.html). The data can be viewed using the BRAT Rapid annotation tool available at http://brat.nlplab.org/ introduced by Stenetorp et al. 2012. 

Stenetorp, P., Pyysalo, S., Topic, G., Ohta, T., Ananiadou, S., and Tsujii, J. (2012). Brat: a web-based tool for NLP-assisted text annotation. In Proc of EACL Demo Session, pages 102–7.

In addition, this release includes the evaluation software used in CLEF e-Health 2015 Task 1b:  

- eval folder: brateval evaluation tool with specific functionalities developped for CLEF e-Health 2015 Task 1b, brateval-ehealth-2015-02-20.jar
